## 🌟 Description

**Combat Coders** is an interactive, game-based learning system that is designed to improve player’s coding skills while also enhancing strategic and decision-making abilities. The game incorporates elements of adventure, card and path-based exploration, situational problem-solving, that creates an interactive and engaging environment where players progress by solving coding challenges. By incorporating programming and developing into a game, Combat Coders aims to make learning more enjoyable, immersive, and effective for students, beginners, and enthusiasts in coding.

## 🔑 Key Features

# 🎮 Combat Coders

## # GAME MODES
* **Adventure Mode** – Battle through distinct levels: Forest, Dungeon, and Volcano.
* **Quest Mode** – Conquer 25 specialized HTML/CSS/JavaScript coding quests.
* **Learn Modules** – Interactive modules with short quizzes for enhanced topic mastery.
* **Character Selection** – Choose your hero: Witch, Knight, or Rogue.

---

## # PLAYER SYSTEM
* **User Authentication** – Secure Login and Registration using JWT tokens.
* **Profile Management** – Custom usernames and a dedicated avatar upload tool with zoom/drag editing.
* **Avatar Presets** – 5 adorable preset options: Penguin, Raccoon, Kitty, Koala, and Cat.
* **Random Avatar Assignment** – New players are instantly assigned a random preset avatar upon signup.
* **Player Stats** – Real-time tracking of Level, XP bars, and account details.

---

## # ACHIEVEMENTS & PROGRESS
* **25 Quest Badges** – Earn a unique badge for every quest completed (badge1 through badge25).
* **3 Level Medals** – Special honors for completion: Forest Guardian, Dungeon Conqueror, and Volcanic Champion.
* **XP System** – Earn experience points dynamically by completing quests and clearing levels.
* **Level Progression** – Advanced leveling mechanics based on total XP earned.

---

## # LEADERBOARD
* **Adventure Leaderboard** – Ranked by the fastest completion times.
* **Quest Mode Leaderboard** – Ranked by the highest player level achieved.
* **Panel Switcher** – Interactive arrow navigation to toggle between leaderboard categories.
* **Rank Display** – The Top 3 players receive special rank icons (`rank1.png`, etc.), while others receive standard ranking.
* **Unranked State** – Players who haven't completed the game are clearly displayed as "Unranked."

---

## # UI/UX FEATURES
* **Pixel Art Style** – Immersive aesthetic using pixelated fonts like *Silkscreen* and *Pixelify Sans*.
* **Responsive Design** – Fully optimized layouts for Mobile, Tablet, and Desktop.
* **Plasma Navbar** – A consistent, sleek navigation bar across all application pages.
* **Learn Dropdown** – Quick-access menu for Game Mechanics and HTML Modules.
* **Settings Panel** – Full control over sound, password changes, and account deletion.
* **Custom Scrollbars** – Themed scrollbars designed to match the retro pixel aesthetic.
* **Animations** – Dynamic leaderboard transitions and particle effects on profile pages.


## 🛠️ Technologies Used

* **Frontend:** VS Code, HTML5, CSS3, JavaScript, Bootstrap, Render
* **Backend:** VS Code, Node.js, Express.js, Mongoose, JWT Authentication, bcrypt, Google OAuth2, Brevo, Render
* **Database:** MongoDB
* **Styling:** Bootstrap, Google Fonts, Piskel, Krita  


## 📥 Installation instructions

**Clone Repository:**
```
git clone https://github.com/hafsaharbotuzon24-lgtm/final-project-Team_Plasma-BSIT2a.git
cd final-project-Team_Plasma-BSIT2a
```

**Install Backend Dependencies:**
```
cd backend
npm install
```

**Install Frontend Dependencies:**
```
cd ../frontend
npm install
```

**Configure environment variables:**
```
cd backend
cp .env.example .env
# Edit .env with own configuration
```

**Start Servers:**
```
cd backend
npm run dev / npm start
```
